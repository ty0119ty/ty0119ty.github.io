MCP(Model Context Protocol)

MCP(模型上下文协议)是 Anthropic 在 2024 年底推出的一套让 LLM 调用外部工具/数据源的标准协议。
你可以把它理解为 “给 LLM 接 USB 设备“ 的统一接口——所有工具都按 MCP 协议暴露,LLM 就能即插即用,不用每个工具单独写适配。

MCP 里的两个角色

  • MCP Host:调用方,通常是 LLM / Agent(比如 Claude Desktop、Langchain4j 应用)
  • MCP Server:被调用方,把”工具/资源”按 MCP 协议暴露出来,跑在本地或远端

MCP 关键点回顾:
