Error能不能滚开

发表于2025-09-17|更新于2025-10-16|技术Java
|浏览量:

开发时遇见的报错

java: java.lang.NoSuchFieldError: Class com.sun.tools.javac.tree.JCTree$JCImport does not have member field ‘com.sun.tools.javac.tree.JCTree qualid’

这样试试看:

  1. 打开终端
  2. java -version
  3. javac -version
  4. 看看两个的版本是否一致
  5. 打开项目结构,看看SDK是不是和java版本相同
  6. 不同改成一样的就行

tm的今天这个java(c) -version是11,但是SDK是21,找半天找不到为什么,问AI还说是什么不兼容,
nm直到用maven的clean弹出来什么21不存在才发现项目结构里面的不一样,wccl

WARN o.s.w.s.m.m.a.ExceptionHandlerExceptionResolver - [logException,208] - Resolved [cn.hutool.json.JSONException: A JSONArray text must start with ‘[‘ at 1 [character 2 line 1]]

这是Redis的报错,是hutool工具的逻辑无法将reids里面的数据转化成对象
希望reids开头是[开头,但是存入的不是,这个时候我们将java代码里面的RedisTempLate加上泛型,
变成RedisTempLate<String,String>这样,就解决了问题了

我也真是服了都,这个报错卡我半天,到时候别来恶心我了,跟着教程一起做,但是就是无法完成,
我也是真的笑了,还是问AI有帮助些,最后自己把泛型加上了
