Blog之Markdown文件
发表于2025-08-12|技术Blog
一些基本符号的应用不同的”#”个数会有不同的标题效果 这是一个”#”号(类似于一级标题)一个 这是两个”#”号(类似于二级标题)两个 使用一个”-“,前后分别加一个空格会一点一点的 就像这样 三个”-“就是一个分割线 小于符号”>”我也不知道叫什么 反正是这样的样子 还有这个”=”和”-“也能区别标题等级,但是只有一级二级这是粗体文字使用两个”*”号 这是斜体文字使用一个”*”号 这是粗斜体使用三个”*”号 两个”~”代表删除样式 (好像不行耶) 这个”<”u”>”表示下划线(中间不要加” 就行) 还有脚注 ^1 一个`可以部分表示代码耶,比如:↓ 使用 git commit 命令提交代码 还有能嵌套的列表 这就是第二个 第三个 没完成,方括号里面不加东西 已完成,方括号里面加一个x 还有好多之后再来探索吧,要睡觉咯 学习网址: https://www.runoob.com/markdown/md-tutorial.html
